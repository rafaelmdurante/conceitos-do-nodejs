# Desafio #1

Desafio #1 do curso **Rocketseat Ignite Node.js**.

Conceitos básicos de `NodeJS`.

## Pré-requisitos

* Node
* Yarn

## Comandos

Todos os comandos a partir do diretório raíz do projeto.

### Instalar Dependências

```bash
yarn install
```

### Rodar Localmente

```bash
yarn dev
```

### Testes

```bash
yarn test
```
